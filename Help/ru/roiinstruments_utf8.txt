<b>
ROI instruments</b><br>
_____________________
<br><br>

General.<br><br>

&nbsp; &nbsp; &nbsp; &nbsp; &nbsp; &nbsp;Данное инструментальное окно предназначено для маркировки выбранных <br>
объектов (ROI - region-of-interest) на изображении с помощью прямоугольников произвольной формы. Этот набор <br>
инструментов полезен при создании датасетов, например для машинного обучения. Прежде чем использовать эти <br>
инструменты, вы должны создать альбом с изображениями для работы. Затем выделите одно из изображений в альбоме<br>
и нажмите кнопку  "инструменты ROI". После загрузки окна вы должны создать новый или загрузить имеющийся ROI файл.<br>
Этот файл имеет текстовый формат и предназначен для хранения геометрии всех созданных ROI, привязанных к конкретным<br>
изображениям.

&nbsp; &nbsp; &nbsp; &nbsp; &nbsp; &nbsp;Маркировка происходит с помощью прямоугольного инструмента, который позволяет<br>
выбрать прямоугольную рамку в нужном месте на изображении, там где находится объект представляющий интерес (ROI). <br>
Начальные размеры инструмента могут быть указаны в специальном боковом меню и могут быть изменены на лету в процессе<br>
маркировки. После применения инструмента вся геометрия записывается в специальный ROI файл, имеющий формат txt. <br>
Обратите внимание, что альбом не привязывается явно к ROI файлу, и этот файл не загружается автоматически при выборе<br>
альбома. Вы должны загрузить этот файл вручную в начале работы.


<br><br>

Создание или загрузка файла ROI.
<br><br>
&nbsp; &nbsp; &nbsp; &nbsp; &nbsp; &nbsp;Для начала создайте новый ROI файл или загрузите уже имеющийся для продолжения работы.<br>
Используйте соответствующие кнопки в верхней части главного тулбара. Обратите внимание что для продолжения сессии работы, когда<br>
ROI файл уже был создан, вы должны каждый раз загружать его вручную при старте окна инструментов ROI.<br>

<br><br>

ROI types.
<br><br>
&nbsp; &nbsp; &nbsp; &nbsp; &nbsp; &nbsp;Для более продвинутой маркировки вам, возможно, будет необходимо различать разные типы объектов.<br> 
Например, при маркировании пешеходов полезно ставить различные метки для велосипедистов, людей которые сидят и т.д. <br>
Вы можете это сделать добавив различные типы объектов в соответствующей таблице внизу экрана. Активируйте выпадающее меню и <br>
выберите пункт "Добавить". После этого в таблице появится новая строка с именем типа "new", который вы можете изменить на желаемый.<br>
При использовании инструмента маркировки вы можете менять тип для текущего ROI с помощью кнопки "E". <br>


<br><br>

Таблица с текущими ROI.
<br><br>
&nbsp; &nbsp; &nbsp; &nbsp; &nbsp; &nbsp;Данная таблица позволяет просматривать список текущих ROI для текущего изображения и<br>
при необходимости удалять ненужное. Также выделение определенной строки в данной таблице позволяет задействовать инструмент<br>
редактирования ROI. При таком выделение ROI будет выделено на изображении другим цветом.
 
<br><br>

Кпнопки навигации по изображениям: next, prev, fisrt unlabeled.
<br><br>
&nbsp; &nbsp; &nbsp; &nbsp; &nbsp; &nbsp;Используйте эти кнопки для перемещения по изображениям альбома.<br> 
Кнопка "Перейти к первому без ROI" позволяет перейти по альбому к первому изображению, которое еще не содержит сохраненных<br> 
ROI, что полезно при продолжении работы. 
<br><br>

Инструмент маркировки ROI.
<br><br>
&nbsp; &nbsp; &nbsp; &nbsp; &nbsp; &nbsp;Данный инструмент собственно позволяет создать новый ROI прямоугольного вида и разместить <br>
его на изображении. При размещении можно зайдействовать клавиши W, A, S, D, для изменения размера рамки ROI. <br>
Использование кнопки "E" позволяет изменять тип текущего ROI, при условии что хотя бы один тип был введен в таблицу типов.

<br><br>

Окно настроек для инструмента маркировки.
<br><br>
&nbsp; &nbsp; Боковое окно настроек, привязанное к кнопке инструмента, повзоляет устанавливать и сохранять параметры для более удобной<br>
манипуляции инструментом.<br>
- Размеры рамки или окна. Начальный размер прямоугольника ROI.<br>
- Сохранять пропорцию при изменении размера. Если этот флаг включен, то изменение размера ROI через кнопки W, A, S, D или с помощью<br>
колесика мыши будет происходит с сохранением изначальной пропорции.<br>
- Шаг размера. Устанавливает величину шага изменения размера. При сильных манипуляциях размером этот шаг можно увеличить для увеличения скорости.<br>
- Автопереключение на след. изображение. Если ваши изображения содержат только один ROI, то данная опция поможет немедленно переключаться на<br>
следующее изображение после использования инструмента маркировки. <br>
- Постоянная активность. После использования инструмента он немедленно активируется снова, что полезно при маркировании множественных<br>
ROI на одном изображении.<br>

Все введенные в данном окне параметры могут быть записаны в буфер памяти, который имеет круговую структуру. Таким образом, <br>
если во время маркировки вам нужны несколько разных наборов параметров, вы может задать их и записать в буфер, извлекая их затем <br>
по мере необходимости.

<br><br>
Статистика ROI.
<br><br>
&nbsp; &nbsp; &nbsp; &nbsp; &nbsp; &nbsp;Этот инструмент предназначен для сбора различной статистики по ROI. Статистика включает<br>
в себя информацию о количестве, средних величинах, наложении и т.д.


<br><br>

Вырезка и сохранение ROI.
<br><br>
&nbsp; &nbsp; &nbsp; &nbsp; &nbsp; &nbsp;Конечной целью маркировки могут быть не только изображения с отмеченными ROI,<br>
но и сами ROI вырезанные в отдельный набор изображений. Данный инструмент позволяет производить подобные датасеты как<br>
для позитивных изображений (ROI), так и для негативных , которые не содержат ROI, и записывать вырезанные изображения в<br>
различные папки согласно выбранным параметрам.

<br><br>

Редактирование ROI.
<br><br>
&nbsp; &nbsp; &nbsp; &nbsp; &nbsp; &nbsp;Иногда необходимо отредактировать геометрию уже имеющегося ROI или изменить его тип.<br>
Данный инструмент позволяет это сделать, и он похож на инструмент маркировки, с единственным отличием: он активируется и <br>
применяется только к конкретному ROI выделенному в таблице. После применения инструмента измененая геометрии и тип записываются<br>
вместо выбранного ROI.
