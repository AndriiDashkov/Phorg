<b>
Фильтрация изображений</b><br>
_________________________
<br><br>
	

&nbsp; &nbsp; &nbsp; &nbsp; &nbsp; &nbsp;&nbsp;Операция &nbsp;&nbsp;фильтрации &nbsp;&nbsp;предназначена&nbsp; &nbsp;для&nbsp; &nbsp;поиска &nbsp;&nbsp;набора &nbsp;изображений, &nbsp;которые <br>
удовлетворяют ряду свойств: например, общий период, используемые темы. Диалог фильтрации <br>
имеет следующие опции:

<br><br><b>
- опция 'Фильтровать'
<br><br></b>
 &nbsp; &nbsp; &nbsp; &nbsp; &nbsp; &nbsp;-- во всех альбомах
<br><br>	
&nbsp; &nbsp; &nbsp; &nbsp; &nbsp; &nbsp;При &nbsp;выборе&nbsp; &nbsp;этой&nbsp; опции&nbsp; &nbsp;фильтр &nbsp;&nbsp;будет&nbsp; работать&nbsp; &nbsp;по &nbsp;всем &nbsp;имеющимся &nbsp;альбомам.<br>
Результирующий набор изображений откроется в специальном окне, которое имеет ряд функций,<br>
позволяющих работать с отфильтрованными изображениями, например, создать новый альбом с<br>
отфильтрованным набором изображений;

<br><br>
 &nbsp; &nbsp; &nbsp; &nbsp; &nbsp; &nbsp;-- в альбоме ....
<br><br>
 &nbsp; &nbsp; &nbsp; &nbsp; &nbsp; &nbsp;При &nbsp;включении &nbsp;второй &nbsp;&nbsp;опции фильтр&nbsp; будет &nbsp;работать &nbsp;только &nbsp;с &nbsp;текущим &nbsp;альбомом <br>
 и отобразит результаты фильтрации &nbsp;прямо в главной &nbsp;панели &nbsp;просмотра.&nbsp; Эта панель&nbsp; имеет &nbsp;в <br>
 заголовке иконку активности фильтра, которая помогает пользователю понять, что изображения <br>
 в главной панели были отфильтрованы, а &nbsp;значит, их видимое &nbsp;количество может отличаться &nbsp;от <br>
 общего &nbsp;&nbsp;количества &nbsp;изображений &nbsp;&nbsp;в &nbsp;альбоме.  &nbsp;&nbsp;Сброс &nbsp;результатов фильтрации &nbsp;может &nbsp;быть <br>
 осуществлен с помощью соответствующей кнопки на &nbsp;главном тулбаре&nbsp; или через &nbsp;контекстное<br>
 меню.  


<br><br><b>
- темы для фильтра
<br><br></b>
 &nbsp; &nbsp; &nbsp; &nbsp; &nbsp; &nbsp;Данная &nbsp;опция &nbsp;предназначена для &nbsp;выбора набора тем,&nbsp; которые &nbsp;будут &nbsp;использованы<br>
 при &nbsp;фильтрации.&nbsp; В &nbsp;результирующий &nbsp;набор &nbsp;попадут&nbsp; только&nbsp; те &nbsp;изображения, в&nbsp; свойствах <br>
 которых указаны выбранные темы;
 


<br><br><b>
- игнорировать при фильтрации
<br><br></b>
 &nbsp; &nbsp; &nbsp; &nbsp; &nbsp; &nbsp;
Данный набор переключателей позволяет исключить из процесса фильтрации тот набор<br>
данных &nbsp;&nbsp;который &nbsp;&nbsp;неинтересен &nbsp;пользователю.&nbsp; Например &nbsp;если &nbsp;&nbsp;в &nbsp;нужных&nbsp; вам &nbsp;результатах <br>
фильтра &nbsp;могут быть&nbsp; изображения как с&nbsp; активными&nbsp; так &nbsp;и &nbsp;с &nbsp;неактивными &nbsp;закладками, &nbsp;тогда <br>
оставьте опцию &nbsp;'закладки' &nbsp;активной. Чтобы &nbsp;включить набор &nbsp;в фильтрацию, &nbsp;наоборот,снимите <br>
соответствующий флажок для &nbsp;тем, закладок, &nbsp;дат и&nbsp; т.д.&nbsp; Для&nbsp; успешной &nbsp;фильтрации &nbsp;хотя &nbsp;бы <br>
одна опция в данной панели &nbsp;должна быть &nbsp;снята, так&nbsp; как&nbsp; в &nbsp;том случае&nbsp; когда&nbsp; включены&nbsp; все <br>
опции (т.е. выбрано полное &nbsp;игнорирование &nbsp;признаков) фильтр не имеет смысла применять.
<br><br><b>
- дата
<br><br></b>
 &nbsp; &nbsp; &nbsp; &nbsp; &nbsp; &nbsp;Здесь вы можете выбрать период времени. Фильтрация отберет только те изображения, <br>
 которые попадают в выбранный период.
<br><br><b>
- фильтровать по<br><br></b>
 &nbsp; &nbsp; &nbsp; &nbsp; &nbsp; &nbsp; Данный набор опций позволяет выбирать между различными принципами&nbsp; фильтрации. <br>
Например &nbsp;если &nbsp;&nbsp;вы &nbsp;хотите &nbsp;чтобы &nbsp;в &nbsp;результаты &nbsp;попали &nbsp;только &nbsp;изображения&nbsp; с&nbsp; активными <br>
закладками, то включите опцию 'по активным &nbsp;закладкам'. Если же наоборот, &nbsp;вы хотите &nbsp;чтобы <br>
в  &nbsp;результаты  &nbsp; &nbsp;попали  &nbsp; &nbsp;только  &nbsp;изображения  &nbsp; &nbsp;с  &nbsp;неактивными  &nbsp;метками  &nbsp;закладок,  &nbsp; &nbsp;тогда <br>
активизируйте опцию 'неактивные закладки'.
