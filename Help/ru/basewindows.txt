<b>Интерфейс пользователя </b><br>
__________________________
<br><br>

Приложение имеет четыре основных окна:<br><br>

<b>
- список (дерево) альбомов
</b><br>
&nbsp; &nbsp; &nbsp; &nbsp; &nbsp; &nbsp;
Это &nbsp;окно &nbsp;содержит &nbsp;список&nbsp; всех &nbsp;альбомов &nbsp;в &nbsp;виде &nbsp;дерева, &nbsp;&nbsp; которое &nbsp;отображает <br> 
зависимости типа &nbsp;родительский альбом/дочерний &nbsp;альбом. Все &nbsp;операции &nbsp;для этого&nbsp; списка <br> 
доступны через меню 'Альбом' или через контекстное меню;
<br><br>
<b>
- главная панель (окно) просмотра
</b><br>
&nbsp; &nbsp; &nbsp; &nbsp; &nbsp; &nbsp;Эта &nbsp;панель&nbsp; &nbsp;отображает &nbsp;все &nbsp; &nbsp;изображения &nbsp;для &nbsp;загруженного &nbsp;альбома и &nbsp;является<br> 
главным местом, где пользователь работает с &nbsp;изображениями.&nbsp; Содержимое главной &nbsp;панели <br> 
может быть изменено с помощью операции загрузки нового альбома, фильтрации, сортировки,<br>  
редактирования, удаления и пр.;
<br><br>  
<b>
- список изображений
</b><br>
&nbsp; &nbsp; &nbsp; &nbsp; &nbsp; &nbsp;Список&nbsp; &nbsp;изображений в &nbsp;виде &nbsp; &nbsp;таблицы &nbsp;позволяет &nbsp;пользователю &nbsp;получать доступ к<br> 
элементам альбома в более быстрой форме, когда само изображение не имеет значения. Эта <br> 
таблица вторична по отношению к главной панели;
<br><br>
<b>
- список тем
</b><br>
&nbsp; &nbsp; &nbsp; &nbsp; &nbsp; &nbsp;Это окно &nbsp;содержит &nbsp;список&nbsp; всех тем,&nbsp; введенных &nbsp;пользователем. &nbsp;Поддерживаются<br> 
операции редактирования, добавления, удаления и пр. Темы как таковые предназначены для <br> 
создания тематических связей между изображениями в разных альбомах.

<br><br>
&nbsp; &nbsp; &nbsp; &nbsp; &nbsp; &nbsp;
Положение и видимость всех описанных окон могут быть изменены через меню Окна.<br>
Ниже для справки приводится список &nbsp;операций, отражающий &nbsp;взаимосвязи между &nbsp;разными <br> 
окнами, т.е. операции, которые, будучи выполнены в одном окне, влияют на состояние<br> 
 другого окна:
<br><br>
- выделение изображения в главной панели с помощью двойного клика на &nbsp;соответствующей <br> 
строке в списке изображений;<br> 
- загрузка изображений другого альбома через контекстное меню списка альбомов;<br>
- изменение иконки альбома через контекстное меню главной панели;<br>
- быстрая вставка &nbsp;выделенной в списке &nbsp;темы во &nbsp;все &nbsp;изображения, выделенные в &nbsp;главной <br>
панели;<br>
- изменение меток печати и закладок в главной панели приводит к синхронизации меток в <br>
списке изображений.
