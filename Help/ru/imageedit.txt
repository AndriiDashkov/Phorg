<b>
Редактирование изображения</b><br>
_____________________________
<br><br>


&nbsp; &nbsp; &nbsp; &nbsp; &nbsp; &nbsp;В контексте данного приложения&nbsp; редактирование означает&nbsp; не графическое изменение <br>
изображения, а изменение его альбомных свойств. Для графического изменения изображения <br>
можно воспользоваться &nbsp;меню 'Разное->Инструменты'. &nbsp;Для &nbsp;начала &nbsp;редактирования &nbsp;свойств  <br>
воспользуйтесь &nbsp;&nbsp;меню &nbsp;'Изображения &nbsp;-> &nbsp;Редактировать'. &nbsp;Диалоговое &nbsp;окно &nbsp;редактирования <br>
практически &nbsp;&nbsp;идентично &nbsp;окну &nbsp;&nbsp;добавления &nbsp;изображения.&nbsp;Вы &nbsp;можете &nbsp;изменять&nbsp; &nbsp;имя, &nbsp;дату, <br>
комментарии, темы
и т.д.

<br><br><b>
Удаление изображения</b><br>
_______________________
<br><br>

&nbsp; &nbsp; &nbsp; &nbsp; &nbsp; &nbsp;Для &nbsp;удаления &nbsp;изображения &nbsp;выделите одно или несколько &nbsp;изображений с помощью <br>
мыши в главной панели и воспользуйтесь меню 'Изображения -> Удаление'. Диалоговое окно <br>
предлагает следующие опции для настройки операции удаления:
<br><br>
- удалить из текущего альбома
<br><br>
&nbsp; &nbsp; &nbsp; &nbsp; &nbsp; &nbsp;Изображение(я) удаляется только из текущего (загруженного) альбома;
<br><br>
- удалить из всех альбомов
<br><br>

&nbsp; &nbsp; &nbsp; &nbsp; &nbsp; &nbsp;  &nbsp;Изображение &nbsp;будет &nbsp;удалено &nbsp;из &nbsp;всех &nbsp;альбомов (одно &nbsp;изображение может&nbsp; быть <br>
вставлено как ссылка в &nbsp;несколько альбомов). При &nbsp;поиске &nbsp;в &nbsp;альбомах &nbsp;данная &nbsp;операция <br>
устанавливает идентичность изображений по совпадению полного пути к файлу;
<br><br>
- не удалять изображение с диска
<br><br>

&nbsp; &nbsp; &nbsp; &nbsp; &nbsp; &nbsp;При выборе этой опции удаляется из альбома только ссылка, сам файл изображения<br>
остается на диске;
<br><br>
- удалять изображения с диска
<br><br>
&nbsp; &nbsp; &nbsp; &nbsp; &nbsp; &nbsp;В &nbsp;данном случае &nbsp;файлы изображений будут &nbsp;физически &nbsp;удалены &nbsp;с диска. Обратите<br>
внимание, что удалены будут только те файлы, которые находятся в области &nbsp;ответственности<br>
приложения, т.е. в &nbsp;стандартной папке &nbsp;альбома. Для файлов &nbsp;изображений, добавленных&nbsp; как <br>
ссылки и не находящихся в стандартном месте хранения, удаления не произойдет.<br><br>

&nbsp; &nbsp; &nbsp; &nbsp; &nbsp; &nbsp;Операция &nbsp;удаления &nbsp;позволяет &nbsp;удалять группу изображений - для этого &nbsp;используйте <br>
мультивыделение в главной панели с помощью мыши при нажатой клавише Shift.
