<b>
Сортировка изображений</b><br>
_________________________
<br><br>

&nbsp; &nbsp; &nbsp; &nbsp; &nbsp; &nbsp;Операция сортировки выполняется с помощью соответствующего выпадающего списка,<br>
расположенного на главном тулбаре. Всего имеется пять видов сортировки:
<br><br>
- по дате<br>
&nbsp; &nbsp; &nbsp; &nbsp; &nbsp; &nbsp; Изображения сортируются согласно свойству 'Дата';
<br><br>
- по порядку вставки<br>
&nbsp; &nbsp; &nbsp; &nbsp; &nbsp; &nbsp; Изображения сортируются согласно очередности вставки в альбом;
<br><br>
- пользовательская сортировка<br>
&nbsp; &nbsp; &nbsp; &nbsp; &nbsp; &nbsp;Изображения &nbsp;сортируются согласно &nbsp;порядка, &nbsp;который &nbsp;пользователь &nbsp;устанавливает <br>
самостоятельно. Этот порядок может быть &nbsp;установлен &nbsp;с помощью &nbsp;операции перетаскивания.<br>
Для активизации режима перетаскивания выберите в ниспадающем списке пользовательскую <br>
сортировку &nbsp;и&nbsp; начинайте &nbsp;&nbsp;перетаскивать &nbsp;&nbsp;изображения &nbsp;прямо&nbsp; &nbsp;на &nbsp;главной &nbsp;панели. &nbsp;После <br>
окончания процесса переупорядочивания выполните сохранение альбома - система сохранит<br>
созданный вариант сортировки.  Обратите &nbsp;внимание, что операция &nbsp;перетаскивания &nbsp;доступна<br>
только для данного режима сортировки;

<br><br>
- по метке печати<br>

&nbsp; &nbsp; &nbsp; &nbsp; &nbsp; &nbsp;Изображения &nbsp;сортируются &nbsp;согласно &nbsp;метке &nbsp;печати - &nbsp;изображения с данной меткой <br>
оказываются в верхней части главной панели;
<br><br>
- по закладкам<br>

&nbsp; &nbsp; &nbsp; &nbsp; &nbsp; &nbsp;Изображения сортируются согласно активным закладкам.<br>
